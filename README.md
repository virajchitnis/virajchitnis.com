virajchitnis.com
================

Built using ReactJS, Webpack, Node.js, ExpressJS and SASS.

Requirements
------------

- Node v6.10.1
- NPM v3.10.x

Develop
-------

yarn start

Simply build assets
-------------------

yarn build

Deploy
------

yarn deploy

Stop
----

yarn stop
